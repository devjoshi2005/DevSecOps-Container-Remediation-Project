Executive Summary

This project outlines a comprehensive plan for a sample cloud security project designed to protect a finance-related application from Distributed Denial-of-Service (DDoS) attacks and facilitate advanced threat analysis. The architecture leverages a hybrid approach, combining the robust, scalable infrastructure of GCP with specialized security services from industry-leading providers.

Key Objectives

  • Implement multi-layered DDoS mitigation using Cloudflare and GCP Cloud Armor
  • Integrate advanced threat analysis with Palo Alto WildFire
  • Ensure PCI DSS compliance for payment processing
  • Establish comprehensive logging with Splunk SIEM

Perimeter Security Layer

Cloudflare Protection

Global anycast network with 100+ Tbps DDoS capacity, WAF with OWASP rules, bot management, and rate limiting.

GCP Cloud Armor

Secondary WAF with custom security policies, geographic filtering, and reCAPTCHA Enterprise integration.

Application & API Layer

Global HTTPS Load Balancer

Single anycast IP with SSL/TLS termination, health checks, and seamless Cloud Armor integration.

API Gateway

Managed Envoy-based proxy with OpenAPI support, authentication, and request transformation.
